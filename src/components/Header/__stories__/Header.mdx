The Header component provides a flexible structure for creating custom headers in web applications.

## Usage

Below is an example of how to use the Header component:

```jsx
import React from 'react';
import { Header } from 'header-component-library';

export const MyComponent = () => {
  const LeftComponent = () => <div>Left</div>;
  const CenterComponent = () => <div>Center</div>;
  const RightComponent = () => <div>Right</div>;

  return (
    <Header
      leftComponent={<LeftComponent />}
      centerComponent={<CenterComponent />}
      rightComponent={<RightComponent />}
    />
  );
};
```
